Detailed Description

Implementation of SVC handler allowing for generic software-pended exceptions and select NEWLIB required and other custom syscalls.

Function Documentation

◆ SVC_C_Handler()

void SVC_C_Handler ( void *  psp)

Takes the provided stack pointer and retrieves the value of the PC (next instruction to execute in user space). Uses the PC to retreive the SVC instruction with the svc_num immediate value. Calls the appropriate syscall implementation depending on the svc_num in the svc instruction.

◆ syscall_exit()

void syscall_exit ( int  status)

Called when a user-space program returnss or terminates from an error. Prints status message with status value and turns on error LED. Sits in an infinite while loop waiting for interrupts as a failsafe error check.

◆ syscall_sbrk()

void* syscall_sbrk ( int  incr)

Manages the current system break in the heap as allocaed by the linker script (using __heap_base as the initial system break value). The system break value is maintained as a static variable in this file (so it is preserved between system calls). Returns either a void pointer to the previous system break value if successful or (void *) -1 on failure.
