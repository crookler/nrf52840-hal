Detailed Description

Contains all of the memory mapped registers necessary for completely specifying the configuration of a sequence (either sequence 1 or sequence 0). Registers are in the correct order, and this struct should be instantiated at the address of the sequence pointer (i.e. PWM_SEQ_BASE_ADDR for PWM_0)
