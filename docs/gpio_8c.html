Implementation of the gpio_init function stub in gpio.h. Takes arugements for port number, pin number, input/output direction, pull/no pull, and drive type. Checks if the pin number if valid for the provided port, and if valid, writes to the appropriate CNF register using memory mapped addresses. Macro definitions match value definitions in CNF register, but additional shift constants are needed to move values to correct offset in CNF register.
