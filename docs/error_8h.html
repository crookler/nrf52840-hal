Detailed Description

Constants for unified error codes across all peripherals. Each error code specifies exactly one unique error.

Macro Definition Documentation

◆ GPIO_INVALID_PORT_ERROR_CODE

#define GPIO_INVALID_PORT_ERROR_CODE   5

Return value of any function returning a numeric type if the pin is out of range for the given port (i.e. >15 for P1 and >31 for P0). Must be positive to match uint8_t return type.
